Formattare e colorare il log di git


git config --global alias.lg "log --color --graph --pretty=format:'%Cred%h%Creset -%C(yellow)%d%Creset %s %Cgreen(%cr) %C(bold blue)<%an>%Creset' --abbrev-commit"
Il comando sopra aggiunge un alias a git, d'ora in avanti posso scrivere: 

git lg
per ottenere un versione del log colorata e formattata in questo modo: Se voglio conoscere i file modificati da ogni commit, posso usare il parametro --stat 

git lg --stat
